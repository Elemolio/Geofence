# Geofence App

Aplicación móvil que notifica al usuario cuando entra o sale de una zona geográfica definida.

## Características

- Definición de geocercas con coordenadas y radios específicos.
- Notificaciones en tiempo real al cruzar los límites geográficos.
- Visualización de la ubicación actual en un mapa.

## Tecnologías utilizadas

- Android SDK
- Google Maps API
